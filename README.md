## 注意事项

请使用 pnpm，如果您使用`npm install`来安装依赖，有可能导致运行错误。
如遇上述情况可以删除 `node_modules`文件夹，再运行`pnpm install`

## 安装 pnpm

npm i pnpm -g

## 安装所有依赖

pnpm install

## 本地测试

pnpm run exm:dev

## 打包

pnpm run build

### 发布

#### 发布到测试库

pnpm publish:dev

#### 发布到生产库

pnpm publish:prod
